Hermes Messenger

Приватный мессенджер с голосовым ИИ. Ваши данные — на вашем оборудовании.

Офлайн-режим

Сообщения сохраняются локально и синхронизируются при появлении связи. Ничего не теряется.

Голосовой ввод

Говорите — сервер распознаёт речь через whisper.cpp и озвучивает ответ через Piper.

Любой ИИ

Работает с Ollama, vLLM, LM Studio, OpenAI — любым API, совместимым с OpenAI.

Приватность

Данные не покидают ваш сервер. Трафик идёт через зашифрованные туннели.
