# Контент-стратегия: Hermes Messenger — Reels / Shorts / TikTok

## Концепция
**"Свой AI-мессенджер"** — показать, что можно иметь своего AI-ассистента с голосом, который работает 24/7 на твоём сервере, offline, и отвечает голосом. Без облаков, без подписок.

## серия 1: ЯДРО (viral-потенциал)

### Reel #1: "Говори — AI отвечает голосом" (15 сек)
**Фича:** Hold-to-record → AI → TTS ответ
**Сценарий:**
1. Кадр: нажимаешь кнопку микрофона (0-2с)
2. Говоришь: "Какая погода завтра?" (2-5с)
3. AI думает (индикатор "печатает") (5-7с)
4. Голосовой ответ из динамика (7-12с)
5. Текст на экране: "Hermes Messenger — свой AI, свой голос" (12-15с)
**Хэштеги:** #aimessenger #voiceai #selfhosted #privacy

### Reel #2: "Два AI в одном чате" (15 сек)
**Фича:** @hermes vs @mimo переключение
**Сценарий:**
1. Кадр: один и тот же вопрос задаётся двум моделям (0-3с)
2. Скрин: тап на чип @mimo → ответ (3-7с)
3. Тап на чип @hermes → ответ (7-11с)
4. Текст: "Выбери своего AI. Или используй обоих." (11-15с)
**Хэштеги:** #multiagent #ai #chatbot #openai

### Reel #3: "Offline тоже работает" (10 сек)
**Фича:** Offline-first архитектура
**Сценарий:**
1. Кадр: airplane mode ON (0-1с)
2. Пишешь сообщение (1-3с)
3. Сообщение отправляется, статус "pending" (3-5с)
4. Airplane mode OFF (5-6с)
5. Сообщение доставлено мгновенно (6-8с)
6. Текст: "Offline-first. Как должен работать мессенджер." (8-10с)
**Хэштеги:** #offlinefirst #messenger #reliable

### Reel #4: "Настройка за 5 секунд" (10 сек)
**Фича:** QR-код onboarding
**Сценарий:**
1. Кадр: браузер с QR-кодом (0-1с)

## серия 2: ПОД КАПОТОМ (для tech-аудитории)

### Reel #5: "WebSocket vs Long-poll" (15 сек)
**Фича:** Real-time доставка + fallback
**Сценарий:**
1. Анимация: WebSocket соединение (зелёная линия) (0-4с)
2. Линия рвётся → оранжевая линия (long-poll) (4-8с)
3. Сообщение всё равно доставлено (8-10с)
4. Текст: "2 слоя надёжности. Никаких потерь." (10-15с)

### Reel #6: "Твой сервер доступен отовсюду" (15 сек)
**Фича:** Tailscale Funnel
**Сценарий:**
1. Карта мира с точкой в твоём городе (0-3с)
2. Текст: "Твой сервер. Твоя сеть." (3-6с)
3. Телефон в другой стране → подключение (6-10с)
4. Чат работает (10-12с)
5. "No port forwarding. No DDNS. Just Tailscale." (12-15с)

### Reel #7: "Обновляется сам" (10 сек)
**Фича:** OTA-обновления с проверкой подписи
**Сценарий:**
1. Push-уведомление: "Update available" (0-2с)

3. "Signature verified" → установка (4-7с)
4. Приложение перезапускается с новой фичей (7-10с)

### Reel #8: "3 GPU для одного AI" (15 сек)
**Фича:** Multi-GPU inference (2x AMD + NVIDIA)
**Сценарий:**
1. nvidia-smi / nvtop — 3 GPU работают (0-4с)

## Рекомендации по продвижению

### Частота
- 3-4 Reels в неделю
- Чередовать: viral (#1-4) + tech (#5-8) + daily (#9-12)

### Платформы
- **TikTok** — основная (algorithm best for tech)
- **Instagram Reels** — вторичная
- **YouTube Shorts** — третичная
- **Twitter/X** — tech community (длинные видео + гифки)

### Формат
- 9:16 (вертикальный)
- 10-15 сек для viral, 15-30 для tech
- Субтитры обязательны (80% смотрят без звука)
- Музыка: lo-fi / ambient (не перебивать голос)
